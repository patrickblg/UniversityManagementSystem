About

This is an app for managing a university. It has standard functions like adding to a table, edit, delete and inspecting the object's details. We've also implemented relationships between entities so that the logic behind the entire program is well structured.

Design

The program is mainly written in Java using SpringBoot and Thymeleaf. It has a standard MVC architecture letting the user see what's happening on a browser page. The styling is done using pure css and html.
